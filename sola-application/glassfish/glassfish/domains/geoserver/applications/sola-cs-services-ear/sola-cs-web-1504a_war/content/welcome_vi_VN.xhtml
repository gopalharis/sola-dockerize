Welcome to SOLA Community Server

Using SOLA Community server you can register as Community Recorder and submit claims on behalf of community members. Once you complete registration a confirmation code will be sent on you e-mail address. You will need to activate it, by following the received link. All Community Recorders registrations are subject to review and approve by Community Server managers.

If you are already registered, use Login page to login into the system.

As an unregistered user, you can access information on claims and view map. If you want to submit challenge to the claim, you will need to contact you Community Recorder or enroll as a recorder yourself. Use registration page to proceed with registration.
